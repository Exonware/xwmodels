# Plan Reference — xwmodels

**Library:** exonware-xwmodels  
**Last Updated:** 08-Mar-2026

Lifecycle and planning templates (output of GUIDE_21_PLAN). Additional plans go in `docs/logs/plans/`.

---

## Lifecycle

1. **Ideation** — REF_12_IDEA → REF_22_PROJECT → REF_13_ARCH  
2. **Planning** — PLAN_* in `docs/logs/plans/`  
3. **Development** — Code + CHANGE_* in `docs/changes/`  
4. **Quality** — Tests, benchmarks, QA gates  
5. **Release** — REF_15_API, CHANGELOG, release logs

---

## Plan Types

- **DEV** — Development work  
- **TEST** — Testing initiatives  
- **DOCS** — Documentation  
- **PROJECT** — Requirements/milestones  
- **BENCH** — Benchmarking  
- **MIXED** — Multi-discipline  

---

## Plan Log

See `docs/logs/SUMMARY_PLAN.md` and `docs/logs/plans/` for PLAN_* documents.

---

## Ref vs Log Documents

| Type | Purpose | Location |
|------|---------|----------|
| **REF_*** | Authoritative current state (WHAT) | `docs/REF_*.md` |
| **Log docs** | Evidence, time-ordered (CHANGE_*, PROJECT_*, TEST_*, etc.) | `docs/logs/`, `docs/changes/` |
